# ShieldAI Family

A React Native parental control app that monitors and protects children's interactions with AI chatbots (ChatGPT, Gemini, Claude, Copilot, Perplexity).

## Features

- 🛡️ Real-time monitoring of AI app conversations via AccessibilityService
- 🔍 Prompt injection detection (direct, indirect, multilingual)
- 🇮🇳 India-first: Hindi, Hinglish, Tamil language support
- 🔒 PII detection (Aadhaar, PAN, UPI, phone numbers)
- 👶 Age-appropriate content filtering (8-12, 13-15, 16-17)
- 📊 Weekly safety reports via WhatsApp
- 🚨 Real-time alerts for parents
- 🆘 Panic button for children

## Quick Start

```bash
npm install
npx expo start
```
